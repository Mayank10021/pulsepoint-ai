# PulsePoint AI – ByteSize Sage AI Hackathon

## Overview
PulsePoint AI converts long educational videos into short viral reels automatically.

## Features
- Upload long video
- Detect key moments
- Generate 3–5 reels
- Streamlit web UI

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
